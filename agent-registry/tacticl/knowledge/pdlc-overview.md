# Tacticl PDLC Pipeline

The Tacticl Product Development Lifecycle (PDLC) pipeline runs specialized AI roles in sequence to handle complex software development sparks end-to-end.

## Roles
- **PM**: Defines requirements, user stories, acceptance criteria
- **RESEARCHER**: Deep research on codebase, domain, prior art
- **ARCHITECT**: System design, ADR, technical decisions
- **DESIGNER**: UX/UI spec, interaction design
- **PLANNER**: Ordered, test-first implementation plan
- **IMPLEMENTER**: TDD execution, commits, PR
- **REVIEWER**: Code review, approval gate
- **TESTER**: End-to-end test suite
- **SECURITY_ANALYST**: Security audit, OWASP review
- **TECHNICAL_WRITER**: Documentation, README updates
- **DEVOPS**: CI/CD, deployment, infra
- **RETRO_ANALYST**: Learning capture, knowledge indexing
